To use, Install Python 3.12 and then open CMD and paste

pip install setuptools


This is a BMD-Optimizer made by Yoshi2 that optimizes BMD File size by converting them to signed16 BMD's which saves about 6kb per 1000 vertices. 

To use, basically you edit the path in optimize.bat (currently set to a dummy folder called "C:\Users\replace\this\with\your\paths") to point at the folder you want to be optimized (for example "C:\Users\Modder\TwilightPrincess\GC\BMD_folder" ) and then you run it and it will save the optimized bmd over the existing BMD's, (so don't use this on a folder that has bmd's you don't want converted). This does work on subfolders
